# Qwen2-VL

## 简介

Qwen2-VL 是 Qwen 家族中第一次把视觉理解、视频理解和视觉 agent 能力系统化写成主线的视觉语言模型节点。

## 关键属性

- 类型：视觉语言模型家族
- 代表来源：
  - [Qwen Team - 2024 - Qwen2-VL](../../raw/summary/Qwen%20Team%20-%202024%20-%20Qwen2-VL.md)
- 当前角色：Qwen 从 LLM 走向 VL 与视觉 agent 的关键入口

## 相关主张

- Qwen2-VL 的重点不只是图文问答，而是把文档、视频、视觉定位和视觉交互统一进同一条 VL 叙事。
- `Naive Dynamic Resolution` 与 `M-ROPE` 是其从早期 VL 范式走向更强通用性的结构升级。
- Qwen2-VL 为 Qwen2.5-VL 和 Qwen2.5-Omni 提供了明确的前序路线。
