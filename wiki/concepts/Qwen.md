# Qwen

## 简介

Qwen 是阿里巴巴提出的大模型家族。在当前知识库中，它不再只表示中文与开源语境下的重要 LLM 主线，也表示一个从纯文本 LLM 逐步扩展到 VL、Omni 与 native multimodal agent 的完整系列。

## 相关主张

- `Qwen Technical Report` 对应的是家族起点，Qwen1.5/2/2.5 则把它逐步发展为多尺寸、多语言、长上下文的成熟开放模型系列。
- 从 Qwen2-VL、Qwen2.5-VL 到 Qwen2.5-Omni、Qwen3.5-Omni，可见 Qwen 已不再局限于纯文本 LLM，而是在向 omni 与 native multimodal agent 演进。
- 在当前知识库中，Qwen 最适合作为“单一家族如何跨越 LLM、VL 与 Omni 主线”的总入口。
