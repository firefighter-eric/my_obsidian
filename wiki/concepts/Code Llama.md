# Code Llama

## 简介

Code Llama 是 Llama 家族中的代码专门模型分支。在当前知识库中，它表示开放基础模型向代码生成、补全和长上下文编程任务外溢的路线。

## 关键属性

- 类型：代码大语言模型家族
- 代表来源：
  - [Roziere et al. - 2023 - Code Llama Open Foundation Models for Code](../../raw/summary/Roziere%20et%20al.%20-%202023%20-%20Code%20Llama%20Open%20Foundation%20Models%20for%20Code.md)
- 当前角色：Llama 家族的代码专门分支

## 相关主张

- Code Llama 不是独立重新起炉灶的模型，而是以 Llama 2 为底座继续代码训练与专门化。
- infilling 与长上下文是它区别于通用 chat / base 模型的关键能力。
- 在当前知识库里，Code Llama 是观察“开放基础模型如何衍生代码专门模型”的代表节点。
